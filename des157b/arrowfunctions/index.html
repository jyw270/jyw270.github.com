Add a short description of your function, what it does and where you got it. You can pull it from some other project you have, or you can create a function from scratch that does something simple and straight forward. Don't just use one of Bill and Glenda's examples though.

Named Function

This is a function called hello() and it takes in a name and prints out 'Hello (name)'. I thought of this idea randomly. 

            function hello(name){
                console.log(`Hello ${name}`);
            }

Function Expression

This function rectArea() takes in a width and height and calculates the area by multiplying them togther. I got this function from MDN. 

            const rectArea = function(width, height){
                width * height;
            }

Arrow Function

            const rectArea = (width, height) => width * height
